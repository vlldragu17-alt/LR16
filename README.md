# Лабораторная работа №16: Базовая настройка Django

## Описание проекта
Приложение представляет собой заготовку для интернет-магазина **"Зеленый Слоник"**. 
Специализация: продажа комнатных растений, семян, горшков и грунта.

## Список страниц
1. **Главная (`/`)** — приветствие и навигация по сайту.
2. **Об авторе (`/about/`)** — информация о разработчике.
3. **О магазине (`/shop/`)** — подробности о тематике и товарах.
